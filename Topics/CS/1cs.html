Binary Systems & Hexadecimal

Binary

  • The binary system has a base of 2. This means it can only be represented with the two digits '0's and '1's. Every heading for a binary digit can be given the headings with the base of 2. For example: 20, 21, 22, 23, 24. Usually a binary number can be written like this - 0111001
  • The binary values is the pure language of your machine's CPU

Denary

  • A denary system has a base of 10. These are numbers everyone is familiar with, 1, 4, 129, 19.

Hexadecimal

  • A hexadecimal system has a base of 16. This means that it will be represented by 16 different digits: 1,2,3,4,5,6,7,8,9,A,B,C,D,E,F
  • Hexidecimal systems are used in HTML to represent color codes, MAC adresses, Web adresses, Assembly code & Machine code

Denary

Converting Denary to Hexadecimal

  • A
    (A->10) 10 * 16^0 = 10
    Answer = 10 = 10
  • 5
    5 * 16^0 = 5
    Answer = 5 = 5
  • 38F4
    4 * 16^0 = 4
    (F->15) 15 * 16^1 = 240
    8 * 16^2 = 2048
    3 * 16^3 = 12288
    Answer = 4 + 240 + 2048 + 12288 = 14580
  • ABC
    (C->12) 12 * 16^0 = 12
    (B->11) 11 * 16^1 = 176
    (A->10) 10 * 16^2 = 2560
    Answer = 12 + 176 + 2560 = 2748

Converting Denary to Binary

To convert denary to binary, you should do the following steps

  1. Divide the denary number by 2
  2. Note down the remainder of the division
  3. Get the quotient of the division from step 1, and divide again by 2
  4. Do this until you reach the quotient 0
  5. Look again at your remainder, at write it down reversed
  6. Once you do this, you'll get your binary value

For example, let's say you want to convert 205 10 to its binary value

Division
by 2		 Quotient Remainder
205/2 102 1
102/2 51 0
51/2 25 1
25/2 12 1

3/2 1 1
1/2 0 1

Remember to put the remainder in from bottom to top to get the binary value,

So 205 10 = 11001101 2

Converting Binary to Denary

To convert binary to denary is very simple, every time the digit '1' appears, add the total to the corresponding header.
For example we want to convert 10100111 to denary:

Binary Digit 1 0 1 0 0 1 1 1
Power 2 27 26 25 24 23 22 21 20

Now add the corresponding heading of the digits that have 1 (in bold)
So you'll get --> 27 + 25 + 22 + 21 + 20
= 167 10

Converting Binary to Hexdecimal

For example you are given the binary value 11100011

Convert every 4 binary bits (from bit0) to hex digit:

111000112 = 1110 --> E | 0011 --> 3 = E316

Converting Hexadecimal to Binary

To convert hexadecimal to binary do the following steps

  1. Start by converting the first digit of the hexadecimal number to denary
  2. Then, convert that denary number to binary (scroll up to see how)
  3. Last, put all the digits together

Pretty simple right, let me show you an example with the number F5A 16 :

  1. Start with the last digit, F.
    F --> 15 --> 1111
  2. Next digit is, 5.
    5 --> 5 --> 0101
  3. And the final digit is, A.
    A --> 10 --> 1010
  4. Finally, put them all together. 1111 0101 1010
