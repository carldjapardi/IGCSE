Arrays

Defining Arrays

Array is a data structure that may contain different data types. It is used to contain different data types into one variable.

For example: myArray = [“Bob”, 141, 45, “Boy”, TRUE]

In the example above, the variable myArray has many different data types such as string, boolean and integers.


Understanding Arrays

Arrays are usually assigned to variables. Arrays are usually used to store related information and using arrays is an efficient way to store/use data. They are also a mutable and ordered data structure. This means that you can change the state of the array and you can index it as well.

Let’s say you are called to make an array of latitudes and longitudes. Instead of assigning multiple values to multiple variables like this:

Position1 = 1232323
Position2 = 1231232
Position3 = 929102

You can store all of these values in an array.
Position = [1232323, 1231232, 929102]

Many higher programming languages prefer using arrays above other data types/structures because they are efficient, mutable and ordered.


Shorcuts

You can easily make a range of arrays of numbers by using a colon.

For example, myNumArray = [1:7]


Using Array In Loops

Let’s say you are given a list of cities:
Cities = [“jakarta”, “washington”, “medan”]

And you want to title case all of the cities...how do you do this?
We can use a loop.

FOR City in Cities
City[0] = chr(ord(City[0])+32)
PRINT Cities
Output: [“Jakarta”, “Washington”, “Singapore”]
