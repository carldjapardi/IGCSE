Communication & Internet Technologies

Data Transmission

  • Simplex, half-duplex, full-duplex
    • Simplex data transmisson is a one direction transmission of data through one line (i.e data sent from your computer to your printer)
    • Half-duplex data transmission is a two direction transmission of data, but online one at a time through line. This means that if X is sending data to Y, Y must wait for X before it can send its own data
    • Full-duplex data transmission is a two direction transmission of data happening simultaneously through one line. This means that both X and Y can send data to each other at the same time
  • Serial & parallel data transmission
    • Serial data transmisson (i.e USB) means data transferring through one line or wire. It is slower then parallel transmisson.
    • Parallel data transmission (CPU internals) means data transferring through many lines/wires. This is faster because more data is sent. However, the drawbacks of using parallel transmission is that the data can become jumbled and unsynchronised
    • In case your wonder what does 64bit computer and 32bit computer, now you know. It is the number of lines in the parallel transmission inside the CPU

Error-Checking

  • Parity checking - check whether data has been corrupted or changed.
  • Automatic Repeat Request (ARQ) - check whether data has been correctly transmitted
  • Cheksum - is another way to check whether data has been corrupted or changed

Internet Technologies

  • Internet Protocol (IP) Address
    • Back then, we don't have domain names like google.com, what we have is IP addresses like 178.123.999.1
    • Since most of us can't remember that, the computer brains went into a room to discuss how to solve this, and they come up with domain names like the one we know today
    • So this domain names have a specific IP address
    • Now with your device, your internet service provider will assign your device with an IP address so that you can surf the web too
  • Web Development
    • Websites are written in HTML, CSS, and Javascript.
    • HTML (HyperTextMarkupLanguage) controls the content and structure of the website
    • CSS controls the style of the website hence the name Cascading StyleSheet
    • Javascript control the behavious of the website
    • So usually, your web browser (chrome, safari, edge) will interpret the HTML, CSS, Javascript and display your website correctly. If you want to see how this website is build, you can right-click your mouse, and click the inspect element. Here you will see the code that build this website
  • HTTP
    • HyperTextTransferProtocol
    • You see this in every website you visit. HTTP is just a set of rules that website developers must follow when transferring data across the internet
    • Sometimes you'll see a HTTPS in some website, this means that the website is secure. This usually appears in sites tha will transfer sensitive information
    • https://www.youtube.com/home
    • The protocol used (https), the web server's name (youtube), the type (com), the file name (home)
